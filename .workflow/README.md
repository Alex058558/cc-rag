# .workflow/

短週期工作文件，用於 AI session 交接與任務追蹤。

## 文件說明

| 檔案 | 角色 | 更新頻率 |
|------|------|----------|
| [PLAN.md](PLAN.md) | 技術架構設計與 Phase 規格 | Phase 完成時 |
| [PROGRESS.md](PROGRESS.md) | 各 Phase 完成狀態 checklist | 功能完成時 |
| [HANDOVER.md](HANDOVER.md) | AI session 交接摘要（目前狀態、下一步） | 每次 session 結束 |

## 使用規範

- HANDOVER.md 只保留「接手需要的資訊」，不是歷史紀錄
- PROGRESS.md 只記狀態標記，不放實作細節
- PLAN.md 是長期設計文件，結構變更時同步更新
- 長期有效的教學內容放 `.tutorial/`，不放這裡
